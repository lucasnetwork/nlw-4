# NLW 4
Criação do backend de um serviço de nps feito durante a nlw 4

## Instalação

```
    yarn
        or
    npm stall
```

## Comandos

### Desenvolvimento
```
    yarn dev
        or
    npm run dev
```
### Testes
```
    yarn test
        or
    npm run test
```
### Typeorm
```
    yarn typeorm
        or
    npm run typeorm
```
